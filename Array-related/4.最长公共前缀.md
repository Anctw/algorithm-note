### [4. 最长公共前缀](https://leetcode.cn/problems/longest-common-prefix/)

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

 

**示例 1：**

```
输入：strs = ["flower","flow","flight"]
输出："fl"
```

**示例 2：**

```
输入：strs = ["dog","racecar","car"]
输出：""
解释：输入不存在公共前缀。
```

 

**提示：**

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- `strs[i]` 如果非空，则仅由小写英文字母组成

#### C++

```C++
// 错误解法
class Solution {
public:
    string longestCommonPrefix(vector<string>& strs) {
        if(strs.size() == 1)
        {
            return strs[0];
        }
        for(int i = 1; i < strs[0].size(); i++){
            string subStr = strs[0].substr(0, i);
            vector<int> boo;
            for(int j = 1; j < strs.size(); j++)
            {
            
                int ret = strs[j].find(subStr, 0);
                boo.push_back(ret);
                
            }
            for(int k = 0; k < boo.size(); k++)
            {
                if(boo[k] != 0)
                {
                    return strs[0].substr(0, i-1);
                }
            }
        }
        return "";
    }
};

// 解
""" 思路：求得两个字符串公共最长前缀  更新ans字符串 二层循环条件 j < ans.size() """
class Solution {
public:
    string longestCommonPrefix(vector<string>& strs) {
        string ans = strs[0];
        for(int i=1;i<strs.size();i++){
            for(int j=0;j<ans.size();j++){
                if(ans[j]!=strs[i][j]){
                    ans = ans.substr(0,j);
                }
            }
        } 
        return ans;
    }
};
```
