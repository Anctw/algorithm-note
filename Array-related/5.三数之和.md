### [5. 三数之和](https://leetcode.cn/problems/3sum/)

给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。

**注意：**答案中不可以包含重复的三元组。



**示例 1：**

```
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
解释：
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
注意，输出的顺序和三元组的顺序并不重要。
```

**示例 2：**

```
输入：nums = [0,1,1]
输出：[]
解释：唯一可能的三元组和不为 0 。
```

**示例 3：**

```
输入：nums = [0,0,0]
输出：[[0,0,0]]
解释：唯一可能的三元组和为 0 。
```



**提示：**

- `3 <= nums.length <= 3000`
- `-105 <= nums[i] <= 105`



#### C++

```C++
// 解
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> ans;
        int size = nums.size();
        if (size >= 3) {
            sort(nums.begin(), nums.end());

            for (int i = 0; i < size - 2; ++i) {
                // 如果当前的数已经大于0，后面两个数也一定大于0，和不可能为0
                if (nums[i] > 0) {
                    break;
                }

                // 跳过以避免重复结果
                if (i > 0 && nums[i] == nums[i - 1]) {
                    continue;
                }

                // 目标是找另外两个数，使得它们的和等于 -nums[i]
                twoSum(ans, nums, i + 1, -nums[i]);
            }
        }
        return ans;
    }

    void twoSum(vector<vector<int>>& ans, vector<int>& nums, int start, int target) {
        int l = start;
        int r = nums.size() - 1;

        while (l < r) {
            auto sum = nums[l] + nums[r];
            if (sum < target) {
                l++;
            } else if (sum > target) {
                r--;
            } else {
                ans.push_back({-target, nums[l], nums[r]});

                // 去重
                auto l_val = nums[l];
                while (l < r && nums[l] == l_val) {
                    l++;
                }

                // 去重
                auto r_val = nums[r];
                while (l < r && nums[r] == r_val) {
                    r--;
                }
            }
        }
    }
};
```
